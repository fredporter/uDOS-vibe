# Empire (Commercial Extension)

Empire is the private, paid business extension for uDOS.

Status:
- Private submodule
- Unreleased / in development
- Commercial (paid add-on)

Scope:
- Business/contact management
- Business email/sync services
- Business data processing + enrichment

Notes:
- Empire is intentionally separated from public Core/Wizard paths.
- Features soft-fail if the extension is not present or the system is not capable.

See `../docs/README.md` for repo-level planning and `docs/README.md` for Empire-specific planning.
