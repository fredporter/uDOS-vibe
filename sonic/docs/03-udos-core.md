# uDOS Core (Alpine Linux)

## Base System
- Alpine Linux
- musl libc
- BusyBox
- openrc init

## Interface
- TUI-first (ncurses)
- Optional Tauri thin GUI
- Single fullscreen process
- No window manager

## Responsibilities
- Hardware discovery
- Disk partitioning
- OS installation
- Bootloader management
- Reboot routing
