# Windows 10 Gaming Layer

## Target
- Windows 10 (unactivated)
- Local account only
- No Microsoft login

## Installed Components
- GPU drivers
- DirectX runtimes
- VC++ redistributables
- Epic Games Launcher
- Fortnite

## Disabled
- Cortana
- OneDrive
- Telemetry (best effort)
- Background apps
- Auto-updates

Windows is treated as an immutable gaming console OS.
