# Icon Starter Set

Lightweight mono SVGs for UI prototyping.

Categories:
- ui (check, close)
- nav (arrow-left, arrow-right)
- system (settings, warning)
- media (play, pause)
- emoji (noto-grin, noto-heart)

Notes:
- Emoji SVGs render via the local Noto Emoji font if available.
- Replace/extend with Noun Project assets as needed.
