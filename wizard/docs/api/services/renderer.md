# Renderer Service

Static render pipeline (themes, missions, exports).

## Routes

- `GET /api/renderer/themes`
- `GET /api/renderer/themes/{theme_name}`
- `GET /api/renderer/site`
- `GET /api/renderer/site/{theme_name}/files`
- `GET /api/renderer/missions`
- `GET /api/renderer/missions/{mission_id}`
- `POST /api/renderer/render`
