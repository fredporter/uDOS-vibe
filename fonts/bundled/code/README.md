## Included Fonts

Five programming fonts with distinct typographic personalities:

| Font        | File                           | Purpose                 | uDOS Usage       |
| ----------- | ------------------------------ | ----------------------- | ---------------- |
| **Argon**   | `MonaspaceArgon-Regular.ttf`   | Neutral, system-like    | `.udos` files    |
| **Xenon**   | `MonaspaceXenon-Regular.ttf`   | Mechanical, precise     | `.ucode` files   |
| **Krypton** | `MonaspaceKrypton-Regular.ttf` | Experimental, italic    | `.wizard` files  |
| **Neon**    | `MonaspaceNeon-Regular.ttf`    | Human-readable, airy    | `.mission` files |
| **Radon**   | `MonaspaceRadon-Regular.ttf`   | Alternative programming | General use      |

---

## Features

- **Texture Healing:** Automatically adjusts letter spacing for better readability
- **Character Variants:** Multiple stylistic alternates per character
- **Ligatures:** Programming-specific ligatures (optional)
- **OpenType Features:** Advanced typography support

---

## Additional Weights & Styles

Only Regular variants are included. To add Bold, Italic, or other weights:

```bash
# Download full distribution
curl -L -o monaspace.zip "https://github.com/githubnext/monaspace/releases/download/v1.301/monaspace-frozen-v1.301.zip"
unzip monaspace.zip
# Copy desired weights from Frozen Fonts/ subdirectories
```
