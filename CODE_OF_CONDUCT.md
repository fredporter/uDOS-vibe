# Code of Conduct

**uDOS Community Code of Conduct**

## Our Commitment

The uDOS project is committed to providing a welcoming, inclusive, and respectful environment for everyone, regardless of:

- Age, body size, disability, ethnicity
- Gender identity and expression
- Level of experience
- Nationality, personal appearance, race, religion
- Sexual identity and orientation
